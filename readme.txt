== Description ==  
Auto-Internal Linker is a WordPress plugin that automatically links predefined keywords to internal URLs, improving SEO and user engagement.  

== Features ==  
✔️ Add unlimited keyword-URL pairs  
✔️ Limit links per page  
✔️ Exclude links from specific posts/pages  
✔️ Click tracking and analytics  
✔️ Security and performance optimized  

== Installation ==  
1. Upload the `auto-internal-linker` folder to `/wp-content/plugins/`.  
2. Activate the plugin via **Plugins > Installed Plugins**.  
3. Configure settings in **Settings > Auto-Internal Linker**.
